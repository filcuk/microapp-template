      Accordion

      A vertical stack of headings that toggle panels below. Only one section stays open at a time. Arrow keys move between headers when focused.

      A vanilla HTML, CSS, and JavaScript starter for small static apps deployed to GitHub Pages.

      Copy markup from demo.html, import the matching module from app/, and call its init helper. See USAGE.md for the full catalogue.

      Use .expand for a single disclosure. Use .accordion when you have several related sections and want one open panel at a time.
